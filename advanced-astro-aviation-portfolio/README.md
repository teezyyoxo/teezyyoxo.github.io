
# Montel Gray Portfolio

A modern Astro portfolio for personal and work-adjacent software projects across
automation, infrastructure, diagnostics, creative tooling, web interfaces, and
aviation-inspired systems.

## Stack

- Astro
- Tailwind-ready configuration
- Static GitHub Pages deployment
- Vanilla JavaScript canvas and GitHub activity widgets

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
In the repository settings, set Pages to deploy from GitHub Actions.
